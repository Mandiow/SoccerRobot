# SoccerRobot
Projeto que usa Lógica Fuzzy para simular um jogador de futebol (Robo quadrado, rústico);

# Projeto
Feito em C++, usando Code::Blocks para compilação. Em virtude do arquivo .cbp não funcionar sempre em outras máquinas, foi excluído do projeto
Para se compilar, abrir um novo projeto e importar todos os arquivos.
